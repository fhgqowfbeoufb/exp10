**AIM**
To study and implement the Pandas library in Python for creating and manipulating Series and DataFrames, and to perform fundamental data handling, exploration, and analysis operations on structured datasets.

**THEORY**
Pandas is an open-source Python library designed for efficient data manipulation and analysis. It introduces two core data structures: **Series**, a one-dimensional labeled array capable of storing heterogeneous data types, and **DataFrame**, a two-dimensional tabular structure composed of rows and columns with labeled axes. These structures enable intuitive handling of structured data similar to spreadsheets or SQL tables.

Pandas supports powerful data access and transformation through label-based and position-based indexing. It also provides built-in statistical and descriptive methods that facilitate quick summarization and exploratory data analysis. Due to its flexibility, performance, and ease of integration with other scientific libraries, Pandas is widely used in data science, machine learning, and real-world data processing tasks.

Key functions and attributes:

* **Series()** – Creates a one-dimensional labeled array.
* **DataFrame()** – Creates a two-dimensional tabular data structure.
* **head()** – Displays the first few rows of a DataFrame.
* **tail()** – Displays the last few rows of a DataFrame.
* **describe()** – Provides statistical summary of numerical columns.
* **mean()** – Computes the average of values.
* **sum()** – Calculates the total of values.
* **max() / min()** – Returns the maximum and minimum values.
* **columns** – Lists column labels of the DataFrame.
* **index** – Shows row labels.
* **loc[]** – Accesses data using row and column labels.
* **iloc[]** – Accesses data using integer index positions.
